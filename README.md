# ☁️ Cloudash

Cloudash is a **cloud-based file management platform** that allows users to upload, organize, and manage files such as images and videos. It provides a structured dashboard where files can be grouped into folders and sections, making it easy to manage media assets for websites or projects.

The platform also allows developers to **connect the storage system to their website**, so uploaded media can be reflected directly on their web applications.

🔗 **Live Demo:**  
https://cloudash-fawn.vercel.app

---

## ✨ Features

- 📁 Create and manage folders
- 🗂 Organize files into sections
- ⬆ Upload images and videos
- 🖼 Preview media files
- ✏ Edit file metadata
- 🗑 Delete files
- 🔍 Search and filter files
- ☁ Cloud storage using Cloudinary
- 🌐 REST API for integration with websites
